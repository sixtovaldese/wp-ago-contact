# aGo Contact

Simple WordPress contact form: Gutenberg block or shortcode, spam protection (honeypot, math captcha, Cloudflare Turnstile), submission inbox in the admin. **Free, no Pro tier.**

[English](#english) · [Español](#espanol)

---

## English

aGo Contact is a focused contact form for WordPress. Activate, configure the email, drop the Gutenberg block or shortcode, done. Submissions go to the admin inbox plus an email notification. Anti-spam includes honeypot, math captcha and optional Cloudflare Turnstile.

### Features

- Gutenberg block and `[ago-contact]` shortcode.
- Configurable fields (name, email, subject, phone, company, department, message, GDPR consent).
- Auto-reply to the sender (optional).
- Anti-spam: honeypot, math captcha, Cloudflare Turnstile.
- Submission inbox in the admin with status, CSV export.
- No external services unless Turnstile is enabled.
- English, Spanish (es_ES) and Brazilian Portuguese (pt_BR) bundled.

### Installation

1. Upload `ago-contact` to `/wp-content/plugins/`.
2. Activate the plugin.
3. Go to **aGo Tools, Contact**. Configure the recipient email.
4. Insert the block or `[ago-contact]` shortcode.
